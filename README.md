# Rust Getting Started

A Rust learning project exploring core language concepts, built from a Scala developer's perspective.

## Topics Covered

- Variables, mutability, and shadowing
- Arrays and iterators
- Ownership and move semantics
- Immutable borrows and shared references
- Mutable borrows and exclusive references
- Guessing game with user input and random number generation

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (installed via rustup)

## Running

```bash
cargo run
```

## Building

```bash
cargo build
```

## Project Structure

```
src/
  main.rs    # All examples and exercises
Cargo.toml   # Dependencies (rand)
```
